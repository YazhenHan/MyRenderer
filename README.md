# MyRenderer
A 3D Mesh Renderer based on OpenGL, including some basic rendering algorithms and mesh processing algorithms.

1. Blinn-Phong shading model
2. Ray tracing
3. Physically based rendering
4. Image based lighting
5. Loop subdivision
6. QEM
